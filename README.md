# Sinatraアプリ

Sinaraフレームワークで構築されたWebアプリケーションとなります。

# 環境準備

①下記、コピー元リポジトリへアクセス
https://github.com/naokiminamikawa/Sinatra-Practice

②< > Codeボタンをクリック -> HTTPSタブで表示されている
下記、URLをクリップボードにコピー
https://github.com/naokiminamikawa/Sinatra-Practice.git

③ローカル環境（任意のディレクトリ上）で以下コマンドを実行
git clone https://github.com/naokiminamikawa/Sinatra-Practice.git

④ディレクトリ（Sinarta-Practice）へ移動
cd Sinatra-Practice

⑤Bundlerを使って必要なGemをインストール
bundle install

⑥ディレクトリ（Sinarta-Practice）上でアプリケーション起動コマンドを実行
bundle exec ruby app.rb

⑦Webブラウザを開き、以下をURL入力し、アプリのトップ画面へアクセス
http://localhost:4567
